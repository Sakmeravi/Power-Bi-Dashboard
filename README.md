# Power-Bi-Dashboard
📌 Project Overview

This project is an interactive Sales Analysis Dashboard built using Power BI to analyze the business performance of Blinkit (an online grocery delivery platform). The dashboard provides insights into sales trends, customer behavior, and product performance.

🎯 Objective
Analyze overall sales performance
Identify top-performing products and categories
Understand customer buying patterns
Provide insights to improve business decisions
📥 Data Source

The dataset used in this project is taken from a public dataset available on Kaggle, which simulates Blinkit’s business operations

🛠️ Tools & Technologies Used
Microsoft Excel → Data cleaning and preprocessing
Power BI → Data visualization and dashboard creation
Power Query → Data transformation (within Power BI)
DAX (Data Analysis Expressions) → Creating calculated measures and KPIs
🔄 Project Workflow
Data Cleaning (Excel):
Removed missing/null values
Corrected data types and formats
Data Import (Power BI):
Loaded dataset into Power BI
Data Transformation (Power Query):
Filtered and structured data
Created new columns where required
Data Modeling:
Established relationships between tables
DAX Calculations:
Total Sales
Average Sales
Total Number of Items
Dashboard Creation:
Designed interactive visuals and charts
Added slicers/filters for dynamic analysis
📊 Dashboard Features
KPI Cards (Total Sales, Average Sales, Total Items)
Sales by Category and Item Type
Outlet-wise Analysis (Location, Size, Type)
Interactive Filters (Slicers)
User-friendly and dynamic visuals
📈 Key Insights
Identified top-selling product categories
Observed sales variation across different outlet types
Found customer preferences and demand patterns
Analyzed peak sales trends.

This dashboard helps in:

The Blinkit Sales Dashboard converts raw data into meaningful insights using Power BI, helping stakeholders understand business performance in a simple and interactive way.
